Overview
This project is designed to demonstrate how basic banking operations can be implemented using Java and object-oriented programming concepts such as classes and methods. The repository currently contains a Java source file named ATMSimulation.java, and the latest commit message indicates that the project implements ATM simulation with account management.

Features
Create and manage bank accounts.

Deposit money into an account.

Withdraw money from an account.

Check account balance.

Simulate basic ATM-style account operations in Java.

Technologies Used
Java.

Object-oriented programming concepts such as classes and methods.
